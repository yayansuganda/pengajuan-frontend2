==============================================
TEMPLATE DOKUMEN PENGAJUAN
==============================================

Folder ini berisi template dokumen untuk pengajuan pembiayaan.

TEMPLATE YANG TERSEDIA:
-----------------------
1. surat-permohonan-anggota.pdf
   - Template Surat Permohonan Anggota & Pembiayaan
   - Format: PDF
   - Penggunaan: Download, isi manual, lalu upload kembali

CATATAN:
--------
- File surat-permohonan-anggota.pdf harus dibuat oleh tim
  dengan format resmi perusahaan
- Template harus mencakup:
  * Kop surat
  * Form data anggota
  * Form data pembiayaan
  * Tempat tanda tangan
  * Kolom persetujuan

CARA MENAMBAH TEMPLATE BARU:
-----------------------------
1. Simpan file PDF di folder public/templates/
2. Update link download di CreatePengajuanWizard.tsx
3. Tambahkan upload field baru di UPLOAD_FIELDS
